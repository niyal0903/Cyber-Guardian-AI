# 🛡️ Cyber Guardian AI

Cyber Guardian AI is an advanced voice-controlled cybersecurity assistant built with Python.
It works like a security-focused AI assistant that can monitor networks, detect suspicious activities, analyze connected devices, and visualize network threats in real time.

The system listens to voice commands and performs cybersecurity operations automatically.

---

# 🚀 Features

### 🎤 Voice Controlled Cyber Assistant

Control cybersecurity tools using voice commands.

Example commands:

- "scan network"
- "system status"
- "packet sniffer"

---

### 📡 WiFi Network Scanner

Scan the local network and detect all connected devices.

Displays:

- IP Address  
- MAC Address  
- Device Name  
- Device Vendor  

---

### 🕵️ Intrusion Detection System

Detect unknown devices connected to your WiFi network and alert the user.

---

### ⚠️ ARP Spoofing Detection

Detect possible ARP spoofing attacks by monitoring MAC address changes.

---

### 🌐 Packet Sniffer

Capture and analyze network packets to monitor suspicious activity.

---

### 💥 DDoS Attack Detection

Detect high packet traffic from suspicious IP addresses and trigger alerts.

---

### 🌍 Hacker Location Finder

Find the approximate location of suspicious IP addresses and open it on Google Maps.

---

### 🔒 Automatic IP Blocking

Block attacker IP addresses using firewall rules.

---

### 📊 Network Visualization

Generate a visual network graph showing connected devices.

---

### 🖥️ SOC Security Dashboard

Displays:

- CPU usage  
- RAM usage  
- Connected devices  
- Threat level
